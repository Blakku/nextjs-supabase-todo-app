# Next.js + Supabase Todo App (Full Stack CRUD)

A full-stack Todo application built with **Next.js** and **Supabase**, demonstrating real-world CRUD operations, database integration, and Row Level Security (RLS).

---

## 🚀 Features

- Create todos (Insert to Supabase database)
- Fetch todos from Supabase
- Delete todos
- Real-time UI updates
- Row Level Security (RLS) enabled
- Clean and minimal UI
- Full working CRUD system

---

## 🧠 Tech Stack

- Next.js (React Framework)
- Supabase (Backend-as-a-Service)
- PostgreSQL Database
- JavaScript (ES6+)
- CSS (Basic styling)

---

## 🔗 Architecture

Next.js Frontend → Supabase API → PostgreSQL Database
